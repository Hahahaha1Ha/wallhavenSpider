# wallhavenSpider

#### 介绍
一个爬取图片的爬虫
图源来自wallhaven.cc
[点我直达](https://wallhaven.cc/)

#### 使用方法
在使用前请先确认自己是否安装了`bs4`,`tqdm`与`requests`库
,如没请使用

```
pip install bs4/tqdm/requests
```
进行下载,
在下载完后就`clone`本仓库
```
git clone https://gitee.com/endlesswuya/wallhaven-spider.git
```
然后进入ImageSpider.py,
在`DataPath=`后输入你保存图片的地址
最后执行即可
